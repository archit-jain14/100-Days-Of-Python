# Day 50: The Halfway Milestone & Core Python Review 🎯

## 📌 Overview
Welcome to **Day 50** of the 100 Days of Code challenge! Reaching this milestone means completing exactly half of the journey. Today’s focus is on reviewing everything learned so far—from basic syntax to advanced file operations—and consolidating backend logic building skills.

---

## 💻 Key Concepts Reviewed Today

*   **Foundational Logic:** Variables, Typecasting, and Conditional Statements (`if-else`, `match-case`).
*   **Loops & Control Flow:** Deep dive into `for` loops, `while` loops, `break`, and `continue` statements.
*   **Data Structures:** Hands-on mastery over Lists, Tuples, Sets, and Dictionaries along with their built-in methods.
*   **Functional Programming:** Writing clean, reusable functions, handling arguments, and understanding Local vs Global scope.
*   **Robust Coding:** Structural Exception Handling using `try-except-finally` blocks and raising custom errors.
*   **File & OS Automation:** Reading/writing files dynamically and automating directory structures using the `os` module.

---
