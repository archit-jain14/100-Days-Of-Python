# 📂 Day 46: Python 'os' Module Automation

A Python automation script developed during my #100DaysOfCode challenge. This project demonstrates how to interact with the Operating System using Python's built-in `os` module.

## 🚀 Features
* **Bulk Directory Creation:** Automatically creates structured folders (`data/Day1` to `data/Day30`) in seconds.
* **File Directory Listing:** Lists all folders inside a specific directory.
* **Bulk Renaming:** Automatically renames folders to a cleaner format.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Core Module:** `os`

## 📄 File Structures
* `main.py`: Script to create the initial folder structure.
* `oslist.py`: Script to list all directories and files.
* `rena.py`: Script to bulk-rename folders dynamically.
