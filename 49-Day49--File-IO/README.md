# 💾 Day 49: File Input/Output (I/O) Automation

A Python script exploring how to interact with the hard drive by reading and writing files programmatically. This is a foundational skill for data handling and automation.

## 🚀 Features & Concepts Covered
* **Reading Files (`read()`):** Opening and pulling data out of `.txt` files.
* **Writing Files (`write()`):** Creating and overwriting data into text files dynamically.
* **Appending Files (`append()`):** Adding new logs/data to existing files without deleting old content.
* **The `with` Statement:** Best practice for file handling that automatically closes files to prevent memory leaks.

## 🛠️ Tech Stack
* **Language:** Python 3
