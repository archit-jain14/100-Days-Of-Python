# 🌐 Day 48: Local vs Global Variables in Python

An educational script demonstrating the concept of Variable Scope in Python.

## 💡 Key Learnings
* **Local Variables:** Variables declared inside a function that cannot be accessed outside.
* **Global Variables:** Variables declared outside any function, accessible globally.
* **`global` Keyword:** Learned how to modify a global variable inside a local function scope.

## 🛠️ Tech Stack
* **Language:** Python 3
